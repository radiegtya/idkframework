# Create Multiple Records

import BrowserWindow from "@site/src/components/BrowserWindow";
import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

In this example we are going to create multiple records to `posts` table

## Request

<Tabs>
  <TabItem value="rest_api" label="REST API" default>

### URL

To <b>create</b> multiple record using REST API, you have to send a <code>POST</code> request to <code>/api/v1/posts/many</code> endpoint

```md title="[POST]"
/api/v1/posts/many
```

### Body

the different with single record is simply pass an array

```json
[
  {
    "title": "Today news is confusing",
    "content": "Our weather today is a little cloudy but the temperature is high"
  },
  {
    "title": "Tomorrow news is better",
    "content": "Our weather tomorrow is a amazing"
  }
]
```

  </TabItem>
  <TabItem value="js_sdk" label="Javascript SDK">
    To send a request using our Javascript SDK, you must install <code>@idk/client</code> from <code>npm</code>

```bash
# npm
npm install @idk/client

# yarn
yarn add @idk/client
```

after installing, you must import <code>IDKClient</code> from <code>@idk/client</code> and use that to **create** multiple records to <code>posts</code> table like below

```javascript
import { IDKClient } from "@idk/client";

const idk = new IDKClient();

const createPosts = async () => {
  const posts = await idk.posts.createMany([
    {
      title: "Today news is confusing",
      content:
        "Our weather today is a little cloudy but the temperature is high",
    },
    {
      title: "Tomorrow news is better",
      content: "Our weather tomorrow is a amazing",
    },
  ]);

  return posts;
};
```

  </TabItem>
</Tabs>
