# Create a Single Record

import BrowserWindow from "@site/src/components/BrowserWindow";
import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

In this example we are going to create a single record to `posts` table

## Request

<Tabs>
  <TabItem value="rest_api" label="REST API" default>

### Url

To <b>create</b> a record using REST API, you have to send a <code>POST</code> request to <code>/api/v1/posts</code> endpoint

```md
[POST] /api/v1/posts
```

### Body

```json
{
  "title": "Today news is confusing",
  "content": "Our weather today is a little cloudy but the temperature is high"
}
```

  </TabItem>
  <TabItem value="js_sdk" label="Javascript SDK">
    To send a request using our Javascript SDK, you must install <code>@idk/client</code> from <code>npm</code>

```bash
# npm
npm install @idk/client

# yarn
yarn add @idk/client
```

after installing, you must import <code>IDKClient</code> from <code>@idk/client</code> and use that to **create** a single record to <code>posts</code> table like below

```javascript
import { IDKClient } from "@idk/client";

const idk = new IDKClient();

const createPost = async () => {
  const post = await idk.posts.create({
    title: "Today news is confusing",
    content: "Our weather today is a little cloudy but the temperature is high",
  });

  return post;
};
```

  </TabItem>
</Tabs>
