# Create Reply

import BrowserWindow from "@site/src/components/BrowserWindow";
import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

Create a record to `Reply` table

## Body

<BrowserWindow url="/replies">
  <h4>Data Transfer Object (DTO)</h4>

```json
{
  "content": "string",
  "postId": "string"
}
```

<p>
  With the DTO above, we are going to create a reply that have a content{" "}
  <code>"That's a really good post"</code> into a post with an id{" "}
  <code>"6c976b04-05dc-4a6e-8154-0c82ecc21b59"</code>
</p>

```json
{
  "content": "That's a really good post",
  "postId": "6c976b04-05dc-4a6e-8154-0c82ecc21b59"
}
```

</BrowserWindow>

## Request

<Tabs>
  <TabItem value="rest_api" label="REST API" default>
  To create a record using REST API, you have to send a <code>POST</code> request to <code>/replies</code> endpoint

```md title="[POST]"
/replies
```

  </TabItem>
  <TabItem value="js_sdk" label="Javascript SDK">
  To create a record using our Javascript SDK, you must install <code>@idk/client</code> from <code>npm</code>

```bash
# npm
npm install @idk/client

# yarn
yarn add @idk/client
```

after installing, you must import <code>IDKClient</code> from <code>@idk/client</code> and use that to create a <code>Reply</code> record like below

```javascript
import { IDKClient } from "@idk/client";

const idk = new IDKClient();

const createPost = async () => {
  const reply = await idk.replies.create({
    content: "That's a really good post",
    postId: "6c976b04-05dc-4a6e-8154-0c82ecc21b59",
  });

  return reply;
};
```

  </TabItem>
</Tabs>
