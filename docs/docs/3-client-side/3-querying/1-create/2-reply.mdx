Create a record to `Reply` table

## Entity

```typescript
export class ReplyEntity {
  id: string,
  content: string,
  postId: string,
  createdById: string,
  updatedById: string,
  createdAt: Date,
  updatedAt: Date
}
```

## Data Transfer Object (DTO)

```typescript
export class CreateReplyDto {
  content: string,
  postId: string
}
```

## Request

<Tabs>
  <TabItem value="rest_api" label="REST API" default>
  To <b>create</b> a record using REST API, you have to send a <code>POST</code> request to <code>/replies</code> endpoint

```md title="[POST]"
/replies
```

  </TabItem>
  <TabItem value="js_sdk" label="Javascript SDK">
  To <b>create</b> a record using our Javascript SDK, you must install <code>@idk/client</code> from <code>npm</code>

```bash
# npm
npm install @idk/client

# yarn
yarn add @idk/client
```

after installing, you must import <code>IDKClient</code> from <code>@idk/client</code> and use that to **create** a <code>Reply</code> record like below

```javascript
import { IDKClient } from "@idk/client";

const idk = new IDKClient();

const createReply = async () => {
  const reply = await idk.replies.create({
    content: "That's a really good post",
    postId: "6c976b04-05dc-4a6e-8154-0c82ecc21b59",
  });

  return reply;
};
```

  </TabItem>
</Tabs>
