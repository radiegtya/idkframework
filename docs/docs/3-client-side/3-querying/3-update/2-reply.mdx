# Update Reply

import BrowserWindow from "@site/src/components/BrowserWindow";
import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

Update a record from `Reply` table

## Data Type Object (DTO)

```typescript
export class UpdateReplyDto {
  content?: string,
}
```

## Body

<BrowserWindow url="/replies/:id">

<p>
  Using our <code>UpdateReplyDto</code>, we are going to update our reply{" "}
  <code>content</code> into <b>"That's a lot of knowledge, thanks!"</b> and{" "}
</p>

```json
{
  "content": "That's a lot of knowledge, thanks!"
}
```

</BrowserWindow>

## Request

<Tabs>
  <TabItem value="rest_api" label="REST API" default>
  To <b>update</b> a record using REST API, you need to have an <code>id</code> from a <code>Reply</code> record, after that send a <code>PATCH</code> request to <code>/replies/:id</code> endpoint

```md title="[PATCH]"
/replies/:id
```

For example, we will <b>update</b> a <code>Reply</code> with an id <code>d9c4bc97-7632-49ba-b1dd-43e7cfdbfc04</code>

```md title="[PATCH]"
/replies/d9c4bc97-7632-49ba-b1dd-43e7cfdbfc04
```

  </TabItem>
  <TabItem value="js_sdk" label="Javascript SDK">
  To <b>update</b> a record using our Javascript SDK, you must install <code>@idk/client</code> from <code>npm</code>

```bash
# npm
npm install @idk/client

# yarn
yarn add @idk/client
```

after installing, you must import <code>IDKClient</code> from <code>@idk/client</code> and use that to **update** a <code>Reply</code> record like below

```javascript
import { IDKClient } from "@idk/client";

const idk = new IDKClient();

const updateReply = async () => {
  const reply = await idk.replies.update(
    "d9c4bc97-7632-49ba-b1dd-43e7cfdbfc04",
    {
      content: "That's a lot of knowledge, thanks!",
    }
  );

  return reply;
};
```

  </TabItem>
</Tabs>
