# update-single-record

import BrowserWindow from "@site/src/components/BrowserWindow";
import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

In this example we are going to update a single record from `posts` table.

## Request

<Tabs>
  <TabItem value="rest_api" label="REST API" default>

### URL

To <b>update</b> a single record using REST API, you need to have an <code>id</code> from a record, after that send a <code>PATCH</code> request to <code>/api/v1/posts/:id</code> endpoint

```md
[PATCH] /api/v1/posts/:id
```

For example, we will <b>update</b> single record from <code>posts</code> table with an id <code>73cd92eb-a530-4b86-bfcd-9efebaa31178</code>

```md
[PATCH] /api/v1/posts/73cd92eb-a530-4b86-bfcd-9efebaa31178
```

### Body

<BrowserWindow url="/api/v1/posts/73cd92eb-a530-4b86-bfcd-9efebaa31178">

```json
{
  "title": "How to be a Software Engineer",
  "content": "You can see some random tutorial at YouTube."
}
```

</BrowserWindow>
  </TabItem>
  <TabItem value="js_sdk" label="Javascript SDK">
  To <b>update</b> a single record using our Javascript SDK, you must install <code>@idk/client</code> from <code>npm</code>

```bash
# npm
npm install @idk/client

# yarn
yarn add @idk/client
```

after installing, you must import <code>IDKClient</code> from <code>@idk/client</code> and use that to **update** a single record from <code>posts</code> table like below

```javascript
import { IDKClient } from "@idk/client";

const idk = new IDKClient();

const updatePost = async () => {
  const post = await idk.posts.update("73cd92eb-a530-4b86-bfcd-9efebaa31178", {
    title: "How to be a Software Engineer",
    content: "You can see some random tutorial at YouTube.",
  });

  return post;
};
```
